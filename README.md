# 🛡️ 银发守护 AI

> 面向老年人的 AI 数字生活安全助手
>
> 帮老人识别诈骗 · 看懂通知 · 完成办事 · 联系家人

---

## 一句话定位

**老人收到一条消息，AI 帮ta判断是不是诈骗、怎么处理、要不要找家属。**

---

## 为什么做这个

- 中国有 **2.97 亿** 60 岁以上老年人
- 每年老年人电信诈骗损失超 **300 亿元**
- 90% 的老年人收到过可疑短信/电话
- 子女无法 24 小时陪在身边

老年人不是不会生活，而是被快速数字化排除在外。银发守护是ta们的 **解释器 + 缓冲器 + 求助桥梁**。

---

## 初赛四大闭环

### 1. 诈骗识别器
粘贴短信/微信话术 → AI 输出风险等级 + 可疑点 + 大白话解释 + 行动建议

### 2. 家属求助卡
一键生成"帮我看看是不是骗子"的转发话术，老人点一下就能发给子女

### 3. 办事陪练
医保报销、医院挂号、养老金认证、快递取件...拆成 3-6 步，一步一步教

### 4. 关怀提醒
用药时间、复查日期、饮食注意，整理成一天的时间表，家属也能查看

---

## 设计原则

| 原则 | 实现 |
|------|------|
| 先给"不要做"再给"要做" | 所有建议遵循安全优先顺序 |
| 每句不超过 25 字 | 强制短句，老人能记住 |
| 零专业术语 | "网页地址"不说"链接" |
| 大字高对比 | 最小 18px，红色=危险，绿色=安全 |
| 大触控区域 | 最小 48×48px，不怕点错 |

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

打开 http://localhost:3000

### 使用真实 AI API

```bash
# 设置环境变量（DeepSeek 为例）
export NEXT_PUBLIC_DEEPSEEK_API_KEY=your_api_key
export NEXT_PUBLIC_DEEPSEEK_BASE_URL=https://api.deepseek.com
export NEXT_PUBLIC_DEEPSEEK_MODEL=deepseek-chat
```

不设置 API Key 时，自动使用内置 Mock 数据——也能完整体验全部功能。
